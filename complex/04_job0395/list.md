
# 研究目标：

1. 基于生物信息学方法，整合脓毒症转录组数据集，筛选针灸调控下与内源性活性物质相关的候选基因，明确针灸干预脓毒症过程中内源性活性物质相关基因的表达特征。

2. 运用机器学习（随机森林、LASSO、SVM-RFE等）算法，对候选基因进行筛选与验证，精准识别出脓毒症中受针灸调控的关键核心基因，构建针灸调控脓毒症的核心基因特征集。

3. 结合孟德尔随机化分析结果与临床样本的表达水平检测，对筛选得到的关键核心基因进行表达验证与因果可靠性验证，最终确定能够反映针灸调控脓毒症效应的稳定核心生物标志物。

4. 对筛选出的生物标志物开展系统的生物信息学深度分析，包括GSEA富集分析、免疫浸润分析、mRNA-TF调控网络构建、列线图模型搭建及潜在药物预测，明确生物标志物在脓毒症病理进程及针灸调控中的作用通路与调控机制，为发掘脓毒症针灸治疗的潜在策略提供理论支撑。

5. 依托单细胞RNA测序数据，解析生物标志物在脓毒症关键细胞类型（如免疫细胞、实质细胞等）中的表达分布与动态变化轨迹，揭示疾病进展中细胞异质性特征，阐明生物标志物通过调控特定细胞类型参与脓毒症发生发展及针灸干预的分子特征与作用模式。

# 方法

1. 文献检索与靶点获取：检索PubMed、CNKI、WebofScience、万方数据库，获取"脓毒症+针灸+内源性活性物质"相关靶点；结合PharmMapper、SwissTargetPrediction数据库，得到针灸干预脓毒症的潜在靶点集合。

2. 转录组数据下载：下载脓毒症公共转录组数据集（如GSE134347），进行数据预处理与标准化。

3. 差异表达分析：筛选脓毒症组与对照组的差异表达基因（DEGs），并与针灸和内源性活性物质靶点取交集，得到交集候选基因。

4. GO/KEGG富集分析：对交集候选基因进行GO功能富集（MF/CC/BP）及KEGG通路富集分析，解析核心生物学功能与通路。

5. PPI网络构建：构建交集候选基因的蛋白互作（PPI）网络，可视化展示候选基因间的蛋白互作关系，分析网络中基因的互作特征，明确基因在整个互作网络中的核心调控地位，为后续核心生物标志物的筛选提供蛋白互作层面的支撑（注：PPI网络仅用于展示基因间互作关系，不用于筛选新的候选基因）。

6. 机器学习关键基因筛选：采用支持向量机（SVM）、随机森林（RF）、梯度提升机（GBM）、极端梯度提升（XGBoost）、朴素贝叶斯（NaiveBayes）算法，筛选并获得Hub基因。

7. 表达水平验证：利用独立脓毒症转录组数据集（如GSE131761），验证关键核心基因的表达差异，确定关键基因。

8. MR分析：对候选生物标志物进行孟德尔随机化（MR）分析，筛选具有显著因果关联的基因，最终确定核心生物标志物。

9. GSEA富集分析：对核心生物标志物进行基因集富集分析，挖掘其参与的关键信号通路。

10. 免疫浸润分析：采用CIBERSORT等算法，分析核心生物标志物与免疫细胞浸润的关联。

11. 列线图构建：基于核心生物标志物表达与临床特征，构建脓毒症预后或针灸响应预测列线图。

12. mRNA-TF调控网络构建：构建核心生物标志物与转录因子（TF）的调控网络，揭示上游调控机制。

13. 单细胞数据下载：下载脓毒症单细胞RNA测序数据集（如GSE157363），进行质量控制与标准化。

14. 单细胞数据预处理与聚类：对单细胞数据进行降维、聚类，划分细胞类型并完成细胞类型注释。

15. 差异细胞类型筛选：筛选脓毒症组与对照组间的差异表达细胞类型。

16. 生物标志物细胞定位分析：解析核心生物标志物在不同关键细胞类型中的表达分布，明确细胞特异性。

17. 拟时序分析：构建关键细胞类型动态变化轨迹，揭示核心生物标志物在疾病进展中的表达动态。

18. 细胞通讯分析：分析关键细胞类型间的通讯模式，阐明核心生物标志物参与的细胞间信号传递。

19. 基因调控网络分析：构建关键细胞类型的基因调控网络，解析核心生物标志物的核心调控作用。

20. 实验验证：通过动物或细胞实验，验证针灸干预对核心生物标志物及下游通路的调控效应。

# 拟解决的关键科学问题：

脓毒症（Sepsis）是由感染触发的全身炎症反应综合征，可快速进展为感染性休克与多器官功能衰竭，是重症医学领域致死率极高的疾病之一，其复杂的病理生理机制与有限的治疗手段给临床带来了巨大挑战。针灸作为传统中医特色外治法，已被证实可通过调控机体内源性活性物质与核心基因表达，在脓毒症的炎症调控与器官保护中展现出潜在价值，但其具体分子靶点与调控网络尚未被系统阐明。内源性活性物质在脓毒症的炎症平衡、免疫调节及器官损伤修复中发挥核心作用，是针灸干预脓毒症的关键媒介，然而其在针灸调控下的动态变化规律、与核心基因的协同作用模式仍不明确。此外，机器学习与单细胞测序技术在脓毒症针灸研究中的应用仍处于起步阶段，借助多组学与人工智能技术挖掘针灸调控脓毒症的生物标志物与作用机制，具有重要的科学探索空间与临床转化价值。

本研究将从以下几个方面进行探究：

1. 通过生物信息学方法对脓毒症公共转录组数据集进行分析，识别脓毒症相关差异表达基因（DEGs）；结合文献检索与数据库挖掘获取针灸干预相关内源性活性物质基因集，将二者取交集得到候选基因。

2. 利用多种机器学习算法（随机森林、LASSO、SVM‑RFE等）对候选基因进行特征筛选，获得与脓毒症及针灸干预相关的hub基因；随后通过独立数据集GSE131761对hub基因进行表达水平验证，确定表达趋势稳健的基因；在此基础上开展孟德尔随机化（MR）分析，以基因表达为暴露、脓毒症为结局，筛选与脓毒症存在显著因果关联且无水平多效性的稳健基因位点，最终确定兼具差异表达特征与因果关联证据的核心生物标志物。

3. 对核心生物标志物开展GSEA富集分析、免疫浸润分析、mRNA-TF调控网络构建、列线图模型搭建及药物预测等系统分析，以阐明其在脓毒症病理进程及针灸干预中的分子特征与调控机制。

4. 采用单细胞RNA测序技术，解析核心生物标志物在脓毒症关键细胞类型中的表达分布、动态变化轨迹及细胞间通讯模式，揭示疾病进展中的细胞异质性与调控规律。

5. 通过动物或细胞实验验证针灸干预对核心生物标志物及下游通路的调控效应，确认生物信息学分析结果的可靠性。旨在发现脓毒症针灸治疗的新型潜在生物标志物与干预靶点，明确针灸调控内源性活性物质及核心基因的分子机制，为脓毒症的临床精准治疗提供重要的理论依据与实验支撑。

# 创新性：

1. 本方案聚焦脓毒症针灸干预研究，在生信分析思路上充分借鉴前沿研究方法，参考多篇相关中科院二区文献（PMID:38404591，PMID:41339491）构建分析框架，其中PMID:38404591（FrontImmunol.2024,IF=5.9）采用单细胞RNA测序与转录组联合分析，结合机器学习筛选脓毒症关键生物标志物，与本研究多组学整合思路高度一致；另一篇相关文献（参考文献：PMID:41339491）通过单细胞与转录组数据挖掘脓毒症生物标志物及分子机制，研究设计与本研究多组学结合思路契合。目前以脓毒症、针灸、Bulk转录组、单细胞转录组和机器学习为关键词进行检索的生信领域报道不足10篇，不存在同类型系统整合多组学与针灸干预机制的研究，研究创新性突出。

2. 所选用的脓毒症转录组数据集（如训练集GSE134347、验证集GSE131761）及单细胞数据集（GSE157363）均有权威文献支撑，数据来源可靠、可信度高。引入独立验证集进行表达水平验证，进一步提升了生物标志物筛选结果的稳健性与可重复性；以脓毒症相关血液/外周血单核细胞样本为样本来源的设计，贴合临床实际，便于后续开展动物/细胞实验及临床验证；充足的样本量可支撑差异表达分析、机器学习筛选、单细胞聚类等多维度分析，保障研究结果的稳定性与可靠性。

3. 本分析方案涵盖多维度研究：含转录组差异表达筛选、GO/KEGG功能富集、PPI网络构建等基础分子分析，单细胞测序数据质控、细胞类型注释、拟时序分析、细胞通讯分析等单细胞研究；同时分析免疫浸润特征、免疫检查点关联，开展mRNA-TF调控网络构建、列线图预测、药物敏感性分析及分子对接，为脓毒症针灸干预的分子机制研究及临床转化提供全面支撑。

# 数据准备

1. 训练集：从GEO数据库（<https://www.ncbi.nlm.nih.gov/geo>）下载GSE134347数据集。该数据集原始总样本量为298例，剔除59例非感染性ICU对照样本后，最终纳入156例脓毒症患者样本与83例健康对照样本进行后续分析（全血样本，平台：GPL17586）；测序类型：表达谱芯片；用途：差异表达基因（DEGs）的鉴定、候选基因筛选及机器学习模型构建【参考文献：PMID:35903199】【注：样本数量以实际下载及质控后为准】。

2. 验证集：从GEO数据库（<https://www.ncbi.nlm.nih.gov/geo>）下载GSE131761数据集。该数据集原始总样本量为114例，剔除33例非感染性休克样本后，最终纳入81例脓毒症患者样本与15例健康对照样本进行后续分析（全血样本，平台：GPL13497）；测序类型：表达谱芯片；用途：Hub基因表达水平验证【参考文献：PMID:34144116】【注：样本数量以实际下载及质控后为准】。

3. 针灸干预脓毒症相关靶点及内源性活性物质基因集：通过PubMed、CNKI、WebofScience、万方数据库检索"脓毒症+针灸+内源性活性物质"相关文献，结合PharmMapper、SwissTargetPrediction数据库预测，获取初始针灸干预脓毒症靶点集合及内源性活性物质相关基因集。

4. 单细胞数据集：从GEO数据库（https://www.ncbi.nlm.nih.gov/geo）下载GSE167363数据集（外周血单核细胞样本，样本及类型：10例脓毒症患者样本与2例健康对照样本，平台：GPL24676）；测序类型：**高通量测序（High-throughput
scRNA-seq）**；用途：细胞类型注释、生物标志物细胞定位及动态变化分析【PMID：34749888】【注：样本数量以实际下载为准】。
