# 项目概述

## 项目背景


牙周炎（Periodontitis，PD）是一种慢性炎症性非传染性疾病，影响牙周组织的各个部位，并造成不可逆的损伤[@dannewitz2021periodontitis]。全球范围内，该病的严重患病率为11%。在晚期病例中，牙周炎会导致牙齿脱落和生活质量下降。牙周炎的病因是多因素的。其病理过程始于龈下牙菌斑生物膜，该生物膜作为始动因子，在易感宿主中引发过度的炎症和免疫反应，最终导致牙周支持组织（包括牙槽骨和牙周韧带）发生不可逆的破坏[@kwon2021current]。慢性牙周炎患者常因局部炎症导致牙槽骨流失，其机制主要包括破骨细胞介导的骨吸收增强，以及成骨细胞介导的骨形成减少。该疾病不仅局限于口腔，还可能增加个体罹患心血管疾病、糖尿病、阿尔茨海默病等全身性疾病的风险，同时与胃肠道疾病及不良妊娠结局相关。牙周炎治疗的主要目标在于控制感染与炎症，阻止牙周组织进一步破坏，并预防牙槽骨的吸收。然而，已丧失的组织，尤其是牙槽骨，其再生能力有限[@chen2022single]。在不同分期的牙周炎治疗中，一个关键区别在于：IV期患者不仅需要控制炎症，还需维持或重建功能性牙列，并在康复治疗的前、中、后期严格执行支持性护理计划。研究显示，与I期牙周炎患者相比，IV期患者在10至30年的随访期间，牙周相关牙齿脱落的风险显著更高（风险比为3.73），同时更易发生病理性牙齿移位及其他功能性障碍[@herrera2022treatment]。这一公共卫生问题，其高患病率加重了全球慢性非传染性疾病的负担[@trindade2023prevalence]。在牙周炎进展过程中，大量免疫细胞通过产生细胞因子和生长因子来激活和调节免疫反应，这些因子影响骨细胞活性[@chen2022single]。随着牙周病机制研究的发展，越来越多的研究认为牙周炎不仅由一种甚至多种致病微生物引起，还与细菌感染后牙周微环境的失衡密切相关[@han2023regulation]。在口腔疾病中，关于代谢产物的研究由来已久。Miller在1980年就提出有机酸等代谢产物对龋病的影响。而代谢紊乱疾病同样是导致牙周炎发生的危险因素。目前关于口腔微生物群落通过代谢作用影响口腔疾病发生的研究尚不完善。对龋病、牙周炎微生物因素及其通过代谢作用的认知不足也导致目前治疗手段单调，无法做到对患病人群的靶向治疗和对高危人群的精准防控。因此，进一步明确这种口腔常见病的发病机制，寻找可行的预测手段，筛查并关注高风险人群，从根源上降低疾病的发生风险是当前亟待解决的临床任务[@wu2023saliva; @cheng2026tryptophan]。

代谢物作为信号分子在调节各种生物功能中发挥着关键作用[@weiss2020metabolite]。细胞间通信（CCC）对于维持细胞、器官及完整系统的功能和稳态至关重要[@song2019cell]。异常CCC已被证明会导致许多健康异常然而，推断代谢物介导的细胞通讯（mCCC）一直具有意义与挑战性。在
mCCC
过程中，由发送细胞产生的代谢物会到达接收细胞，这些代谢物与接收细胞中的传感器蛋白结合，从而触发生物过程的调节，mCCC
可以调节各种细胞功能[@zheng2025mebocost]。由此，牙周组织中mCCC也许发挥重要作用。

牙周炎的发生发展不仅源于病原微生物的直接侵袭，也许更与微环境中细胞间通讯的紊乱密切相关。代谢物作为信号分子在调控骨代谢和免疫稳态中的核心作用正日益凸显[@wu2023saliva; @cheng2026tryptophan]。在牙周炎微环境中，局部代谢物的种类与浓度发生显著改变[@cheng2026tryptophan]，它们可能通过细胞表面转运蛋白或核受体等非经典途径，介导免疫细胞与骨细胞之间的异常对话，从而加剧破骨活性并抑制组织再生。因此，解析牙周炎微环境中的代谢物介导的细胞间通讯，有助于揭示传统蛋白信号网络未能覆盖的致病机制，为开发靶向代谢通路的治疗策略提供理论依据。

## 研究目标

1.  建立牙周炎微环境代谢物通讯的分析框架。旨在利用单细胞转录组的高分辨率优势，结合传统bulk转录组的大样本验证特性，引入并优化专门用于预测代谢物介导细胞通讯的算法（MEBOCOST）。通过二者数据的整合策略，突破传统研究仅关注蛋白质配体-受体相互作用的局限，系统性地绘制牙周炎组织中由代谢物驱动的细胞间信号网络图谱。

2.  识别牙周炎进程中关键的"代谢物-传感器"互作对。研究的目标在于精确锁定牙周炎微环境中发挥核心致病作用的特异性代谢物信号。通过分析单细胞数据中不同细胞亚群的基因表达特征，识别负责产生特定代谢物的"发送细胞"，以及通过细胞表面受体、转运蛋白或核受体感知这些代谢物的"接收细胞"，从而揭示在牙周炎发生发展过程中被异常激活或抑制的代谢通讯通路。

3.  重点解析免疫细胞与基质细胞之间的代谢对话机制。致力于阐明免疫细胞与牙周组织中的基质细胞之间如何通过代谢物进行信息交换。探索这类代谢通讯如何调控局部炎症反应的强度与持续时间，以及如何影响牙槽骨的代谢平衡，从代谢角度解释牙周炎导致组织破坏的深层机制。

4.  筛选具有诊断价值或治疗潜力的代谢信号分子。进一步整合bulk转录组数据与临床指标，筛选出与牙周炎严重程度及进展显著相关的代谢通讯分子。这些关键代谢物或其传感器蛋白，一方面可能作为评估疾病活动性的生物标志物，另一方面也可能成为牙周炎治疗的新靶点，为通过干预代谢通路来控制炎症、促进组织再生提供理论依据。

## 拟解决的关键科学问题

本研究拟解决的关键科学问题是：牙周炎微环境中是否存在独立于经典蛋白信号之外的代谢物介导的细胞间通讯网络，以及该网络如何通过调控免疫-骨代谢对话参与牙槽骨流失的病理过程。这一核心目标可从以下四个层面来解释：

1.  代谢物介导的细胞通讯在牙周炎微环境中是否存在特异性的编码模式。传统观点认为牙周炎的发生发展主要受蛋白质信号分子调控。然而，在炎症微环境中，局部代谢物的种类和浓度常发生剧烈变化，这些代谢物是否作为一种通讯语言在细胞间传递信息尚不清楚。本研究通过整合单细胞与bulk转录组数据，验证牙周炎组织中是否存在独立于经典蛋白配体-受体途径之外的、由代谢物驱动的系统性细胞通讯网络，并揭示其与健康状态下的差异，从而解析代谢物通讯是否参与了牙周炎病理过程这一科学问题。

2.  牙周炎微环境中关键代谢物通过何种传感器蛋白介导了免疫细胞与基质细胞的异常对话。牙周炎的核心病理特征是宿主免疫反应与牙槽骨代谢的失衡。目前尚不明确的是，免疫细胞与骨相关细胞及牙龈成纤维细胞之间，是否存在通过代谢物进行的信号交换。本研究希望通过分析单细胞数据中代谢物或其激酶与受体共表达模式，精确锁定介导这两种细胞类型之间异常通讯的关键代谢物分子，阐明它们是如何调控炎症反应的。

3.  基于单细胞转录组推断代谢物通讯的算法在牙周炎症组织中通过整合bulk转录组数据进行校准与验证。由于代谢物本身无法直接通过测序技术检测，利用转录组推断代谢物通讯面临固有挑战：mRNA表达水平并不能完全代表代谢物的实际浓度与通量，且代谢酶的翻译后修饰及底物可用性均会影响代谢物的产生。因此，利用bulk转录组的大样本统计效力与单细胞数据的高分辨率，对MEBOCOST算法的预测结果进行验证，从而克服单纯依赖基因表达推断代谢物通讯的技术瓶颈，提高识别结果的可靠性。

4.  阐述牙周炎相关的代谢物通讯信号作为疾病诊断标志物或治疗靶点的潜在价值。目前临床上缺乏能够有效反映牙周炎活动性尤其是骨吸收进程的分子标志物。由单细胞数据鉴定的关键代谢通讯分子，验证其在bulk转录组层面的表达特征与牙周炎的临床指标及进展风险存在显著相关性。旨在从复杂的代谢通讯网络中筛选出核心节点分子，并探讨通过靶向特定代谢物或其传感器来阻断病理信号、促进组织修复的可行性。


## 创新性

本研究的创新性在于：突破传统细胞通讯研究聚焦于蛋白质配体-受体的局限，将代谢物介导的细胞间通讯引入牙周炎微环境解析，为理解牙周炎的免疫失衡与骨代谢紊乱提供了全新的信息维度；其次，在方法学上创新性地整合了单细胞转录组的高分辨率与bulk转录组的大样本验证能力，并结合MEBOCOST算法，克服了单纯转录组推断代谢通路的固有偏差。研究致力于系统揭示牙周炎进程中细胞之间的代谢对话网络，有望从代谢调控的角度发现新的干预靶点，为牙周炎的诊断和治疗开辟基于代谢信号的新策略。

本研究方案体现如下：

1.  研究思路是来源于一区和二区文章 (PMID：35154475，PMID: 40568942)。通过整合单细胞转录组的高分辨率细胞分型与bulk转录组的大样本验证，系统解析牙周炎微环境中代谢物介导的细胞间通讯网络，旨在揭示传统蛋白质信号之外调控免疫-骨代谢对话的新机制。

2.  所选用的数据集来自human牙周组织均有文献作为支撑，可信度高。

3.  这项研究体现了多组学整合在解析复杂疾病机制中的方法论革新，通过将单细胞分辨率与bulk转录组广度深度融合，突破了传统细胞通讯研究局限于蛋白质信号的框架。其核心价值在于从代谢维度重新审视牙周炎的病理本质，揭示了代谢物作为信息载体在连接局部炎症与全身性疾病风险中的潜在桥梁作用。

## 集成单细胞数据分析

### 数据质控

以 R 包 `Seurat` ⟦pkgInfo('Seurat')⟧ 进行单细胞数据质量控制 (QC) 和下游分析。依据 <https://satijalab.org/seurat/articles/integration_introduction> 为指导对单细胞数据预处理。前期质量控制⟦ref('GSE171213-before-Quality-control', autoRef = 'fig')⟧，一个细胞至少应有 200 个基因，并且基因数量小于 3000。线粒体基因的比例小于 25%。保留总基因表达量小于 10000 细胞。过滤前，所有样本共包含 37665 个细胞，24540 个基因。过滤后⟦ref('GSE171213-After-Quality-control', autoRef = 'fig')⟧，⟦mark$red('所有样本共包含27414个细胞，24540 个基因用于后续分析。')⟧使用 `Seurat::SCTransform` (默认参数) 对数据集归一化 (<https://satijalab.org/seurat/articles/sctransform_vignette>) 。随后 PCA 聚类 (`RunPCA`)。使用 ElbowPlot 函数绘制肘图⟦ref('GSE171213-Standard-deviations-of-PCs', autoRef = 'fig')⟧，帮助确定用于下游分析的主成分以进行后续分析。结果显示，前 15 个 PCs 以后方差增量减缓逐渐趋于稳定，选择前 15 个 PCs 进行后续聚类分析。以 `Seurat::IntegrateLayers` 集成数据，去除批次效应 (使用 HarmonyIntegration 方法)。在 1-15 PC 维度下，以 `Seurat::FindNeighbors` 构建 Nearest-neighbor Graph。随后在 1.2 分辨率下，以 `Seurat::FindClusters` 函数识别细胞群并以 `Seurat::RunUMAP` 进行 UMAP 聚类。在去除批次效应前，UMAP 图⟦ref('GSE171213-UMAP-Unintegrated', autoRef = 'fig')⟧中各样本保持离散。`Seurat::FindClusters` 共找到 44 个细胞簇。去除批次效应后⟦ref('GSE171213-UMAP-Integrated', autoRef = 'fig')⟧，`Seurat::FindClusters` 找到 32 个细胞簇，且各样本相互均匀混合，即批次效应已被良好地处理。选择去除批次效应后的数据集进行后续分析。以 `Seurat::FindAllMarkers` (LogFC 阈值 0.25; 最小检出率 0.25; 检验方法为 wilcox) 为所有细胞群寻找 Markers。以 Python 工具 `SCSA` [@ScsaACellTyCaoY2020] (<https://github.com/bioinfo-ibms-pumc/SCSA>) 结合特异 Marker 对细胞群注释。
