# 分析结果 {#workflow}

使用 Seurat R 包（5.1.0）进行单细胞数据质量控制（QC）和下游分析。依据 <https://satijalab.org/seurat/articles/integration_introduction> 为指导对单细胞数据预处理。前期质量控制，一个细胞至少应有 200 个基因，并且基因数量小于 6000。线粒体基因的比例小于 20%。保留总基因表达量小于 20000 细胞。过滤后，所有样本共包含98998个细胞用于后续分析。使用 `Seurat::SCTransform` 对数据集归一化（<https://satijalab.org/seurat/articles/sctransform_vignette>）。随后 PCA 聚类（`RunPCA`），绘制 PC standard deviations 图。以 `Seurat::IntegrateLayers` 集成数据，去除批次效应（使用 HarmonyIntegration 方法）。在 1-15 PC 维度下，以 `Seurat::FindNeighbors` 构建 Nearest-neighbor Graph。随后在 1.2 分辨率下，以 `Seurat::FindClusters` 函数识别细胞群并以 `Seurat::RunUMAP` 进行 UMAP 聚类。以 `Seurat::FindAllMarkers`（LogFC 阈值 0.25；最小检出率 0.25）为所有细胞群寻找 Markers。以 Python 工具 `SCSA` [@ScsaACellTyCaoY2020]（<https://github.com/bioinfo-ibms-pumc/SCSA>）结合文献（PMID: 37735465）的 Marker 对细胞群注释。
